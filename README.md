# Tic Tac Toe

A simple implementation of the classic Tic Tac Toe game built using HTML, CSS, and JavaScript. This project demonstrates fundamental web development concepts and provides a fun and interactive way to play the game in your browser.

## Features
- Two-player mode (Player X and Player O).
- Intuitive game interface.
- Dynamic win detection and draw handling.
- Responsive design for different screen sizes.

## Technologies Used
- **HTML**: Structure of the game board and layout.
- **CSS**: Styling and layout adjustments.
- **JavaScript**: Game logic and interactivity.

## How to Play
1. Open the `index.html` file in a web browser.
2. The game starts with Player X.
3. Click on an empty square to place your mark (X or O).
4. The game alternates between Player X and Player O.
5. A player wins by aligning three marks horizontally, vertically, or diagonally.
6. If all squares are filled without a winner, the game ends in a draw.
7. Refresh the page to restart the game.

## File Structure
```
project-folder/
|-- index.html
|-- styles.css
|-- script.js
```
- **index.html**: Contains the game layout.
- **styles.css**: Defines the appearance of the game board and interface.
- **script.js**: Implements the game logic and interactions.

## Setup Instructions
1. Clone this repository or download the project files.
2. Open `index.html` in your preferred browser.
3. Play the game!
